# 📊 Análisis de Datos

Repositorio educativo desarrollado como parte de un bootcamp enfocado en el aprendizaje y aplicación de técnicas de análisis de datos.

## 📌 Descripción

Este repositorio contiene material técnico orientado al estudio y práctica del análisis de datos utilizando herramientas modernas. Su objetivo es documentar el proceso de aprendizaje, así como servir de referencia para futuros proyectos en el área.

## 🛠️ Tecnologías utilizadas

* Python 🐍
* Google Colab 📓

## 📂 Contenido

El repositorio incluye:

* Notebooks con ejercicios prácticos
* Análisis exploratorio de datos (EDA)
* Ejemplos de manipulación y limpieza de datos
* Aplicación de conceptos fundamentales de análisis

## 🚀 Uso

Puedes utilizar este repositorio de la siguiente manera:

1. Clona el repositorio:

   ```bash
   git clone https://github.com/tu-usuario/tu-repositorio.git
   ```

2. Accede a los archivos desde:

   * Google Colab (recomendado)
   * Jupyter Notebook (entorno local)

3. Ejecuta los notebooks paso a paso para comprender el flujo de análisis.

## 🎯 Objetivos de aprendizaje

* Comprender los fundamentos del análisis de datos
* Aplicar técnicas de limpieza y transformación de datos
* Realizar análisis exploratorio (EDA)
* Visualizar datos de forma efectiva
